# Cat and Dog Image Classification using SVM

## Objective
Build an SVM (Support Vector Machine) model to classify images as Cats or Dogs.

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn
- Matplotlib

## Dataset
Microsoft Cats and Dogs Dataset

## Algorithm
Support Vector Machine (SVM)

## Result
The model successfully classified Cat and Dog images.
